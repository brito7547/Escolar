Configurar Supabase

1. Crie uma conta em supabase.com
2. Crie um projeto gratuito
3. Vá em SQL Editor e rode este script:

create table if not exists sala_tarefas ( id uuid default gen_random_uuid() primary key, sala_id text not null, titulo text not null, materia text, autor text, feita boolean default false, criado_em timestamptz default now() ); create index on sala_tarefas(sala_id); alter table sala_tarefas enable row level security; create policy "public" on sala_tarefas for all using (true);

4. Vá em Project Settings → API e copie a URL e a chave anon/public.
